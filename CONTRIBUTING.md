# Contributing to Awesome VC & Startup Open Source

Thank you for helping make this the most comprehensive MECE resource for VC tooling!

## Before You Submit

- The tool is open-source (MIT, Apache 2.0, GPL, BSD, CC0, or similar)
- It has a public GitHub repository
- It is actively maintained (last commit within 2 years) or historically significant
- It fits in exactly one of the 11 categories (MECE rule)
- It is not already listed

## Adding a New Tool

1. Fork this repository
2. Find the right category in README.md and the matching file in categories/
3. Add a row to the table using this format in both files
4. Open a Pull Request with title: Add [repo-name] to [Category Name]

## Stage Tags

| Tag | Stage | When to Use |
|-----|-------|-------------|
| Pre-Seed | Idea stage, pre-revenue, MVP only |
| Seed | Early revenue, PMF testing, first institutional round |
| Series A/B | Scaling, growth metrics established |
| Series C/D | Hypergrowth, market leadership |
| Late Stage | Pre-IPO, large enterprise scale |
| All Stages | Stage-agnostic utility |

## MECE Category Rules

Each tool must fit exactly one category:

1. Deal Sourcing and Pipeline - Finding and tracking investment opportunities
2. Due Diligence and Data Rooms - Evaluating and organizing DD processes
3. Financial Modeling and Valuation - DCF, comparables, runway, burn rate
4. Legal SAFE and Term Sheets - Legal docs, SAFE notes, equity agreements
5. Cap Table and Equity Management - Ownership, dilution, option pools
6. Portfolio Management and Monitoring - Post-investment tracking and dashboards
7. Fund Operations and LP Reporting - Fund admin, waterfall, LP comms
8. Startup Metrics and KPIs - SaaS metrics, unit economics, AARRR
9. Pitch Decks and Fundraising - Pitch creation, analysis, accelerators
10. Market Research and Intelligence - Competitive analysis, market sizing
11. AI and ML Tools for VC - AI-driven investment research and analysis

## Quality Bar

Include tools that solve a real VC/startup problem, have working code, and would help a developer build something useful.

Exclude: Pure datasets, unmaintained repos, purely commercial tools, tutorials without reusable code.

## Reporting Broken Links

Open an issue with title: Broken Link repo-name and include the dead URL.
